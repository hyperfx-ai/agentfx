# B2B Growth Templates

To execute B2B SaaS growth tactics, use the ready-made templates in the reference files. Each template is designed to be copied and customized for your specific product.

## When to Use This Skill

Use this skill when:
- Writing Facebook or Google ads
- Sending cold outreach (email or LinkedIn)
- Setting up paid ad campaigns
- Reaching out to podcast guests
- Creating email nurture sequences
- Building conference booth strategies

## How to Use This Skill

1. **Identify the tactic** you need to execute
2. **Load the appropriate template file** from `references/`:
   - `facebook-ad-scripts.md` - 10 UGC ad scripts (Hook → Pain → Solution)
   - `cold-email-templates.md` - 15 cold email templates for different use cases
   - `linkedin-sequences.md` - 4-message LinkedIn sequences
   - `google-ads-setup.md` - Campaign structure and ad copy templates
   - `podcast-outreach.md` - Guest outreach and newsletter templates
3. **Copy the template** that matches your use case
4. **Customize** with your product details, customer pain points, and value prop

## Core Principles

Before using any template, understand these principles:

**Average product + good marketing beats good product + average marketing**

**The Pattern:**
Value → Value → Value → Value → Value → Value → Ask
(Repeat this rhythm in all channels)

**Testing:**
- Start small
- Test quickly
- Double down on winners
- Don't touch what's working
