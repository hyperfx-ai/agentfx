# LinkedIn Message Sequences

## Template Instructions

1. Each sequence = connection request + 3 follow-ups
2. Wait 3 days between messages
3. Keep messages under 150 characters for connection requests
4. Keep follow-ups under 300 characters
5. Personalize at least one element per message

---

## Sequence 1: The Problem Solver

**Connection Request:**
```
Hey [NAME] - saw you're [ROLE] at [COMPANY]. I work with [SIMILAR_COMPANIES] on [PROBLEM]. Would love to connect and share what we're seeing in the space.
```

**Day 3:**
```
Thanks for connecting! Quick question - how is [COMPANY] currently handling [SPECIFIC_TASK]?

Most teams at your stage are either [OPTION_A] or [OPTION_B]. Curious what's working for you.
```

**Day 7:**
```
No worries if you're swamped. 

Just wanted to share this quick guide on [TOPIC]: [LINK]

[SIMILAR_COMPANY] used this approach to [RESULT]. Thought it might be relevant.
```

**Day 14:**
```
Last note - we help [ROLE]s at [COMPANY_TYPE] [SPECIFIC_OUTCOME].

[CASE_STUDY] went from [BEFORE] to [AFTER] in [TIMEFRAME].

Happy to share the approach if useful. If not, totally cool - good luck with [THEIR_INITIATIVE]!
```

## Automation Strategy

**Tools:**
- HeyReach or Expandi ($40/seat)
- Phantom Buster for list building
- Sales Navigator for targeting

**Limits:**
- 100 connection requests per week per account
- 400 total per month
- Don't message on weekends (low response rate)

**Success Metrics:**
- 30-40% connection acceptance rate = good
- 5-10% reply rate after connection = good
- 1-2% meeting booked rate = good

**Scaling:**
- Start with 1 account
- Test all sequences
- Find winner
- Spin up 5-10 accounts with winner
- 25 accounts = 10,000 outreach/month
